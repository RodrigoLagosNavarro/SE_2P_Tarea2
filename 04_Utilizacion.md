4. Interfaz de Usuario y Subsistema de Explicación

¿Qué?: La interfaz es el enlace entre el sistema y el usuario ; el subsistema de explicación justifica las acciones tomadas.

Para qué?: Para que el sistema sea efectivo, fácil de usar y transparente en sus decisiones.

¿Cómo?: La interfaz muestra conclusiones y pide información faltante ; el de explicación rastrea los pasos lógicos del motor de inferencia.

Ejemplo: Un cajero automático te explica que retiene tu tarjeta porque fallaste la clave tres veces.
