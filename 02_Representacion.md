2. Base de Conocimiento (y Memoria de Trabajo)

¿Qué?: Es donde se almacena el conocimiento (reglas generales) y los datos (información específica).

Para qué?: Para separar el conocimiento permanente (como leyes de medicina) de los datos efímeros de un caso particular.

¿Cómo?: El conocimiento se guarda en la "Base de Conocimiento" y los datos transitorios en la "Memoria de Trabajo".

Ejemplo: El conocimiento dice "la fiebre es un síntoma"; el dato dice "el paciente Juan tiene 38.5°C de fiebre".
