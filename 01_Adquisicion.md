1. Módulo de Adquisición de Conocimiento

¿Qué?: Es el subsistema que controla el flujo de nuevo conocimiento desde el experto hacia el sistema.


Para qué?: Para determinar si la información recibida es nueva y debe incluirse en la base de datos para mantener el sistema actualizado.


¿Cómo?: Los ingenieros del conocimiento trasladan el saber de los expertos humanos a un lenguaje que el sistema pueda entender.

Ejemplo: Un ingeniero traduce las reglas de un médico sobre cómo identificar una gripe para que el software las procese.
